TROUBLESHOOTING
================================================================================

‚ùå Problem: "Something went wrong" error
‚úÖ Solution:
   1. Click "Web" tab
   2. Click "Error log" link
   3. Check what the error says
   4. Common fixes:
      - Make sure you replaced YOUR_USERNAME in WSGI file
      - Make sure paths are correct
      - Check that requirements.txt was installed

‚ùå Problem: Images not showing
‚úÖ Solution:
   1. Check Static files configuration
   2. Make sure /static/ maps to correct path
   3. Click "Reload" button

‚ùå Problem: "No module named 'app'"
‚úÖ Solution:
   1. Check WSGI file has correct path
   2. Check Source code directory is set
   3. Make sure you're in the right directory

‚ùå Problem: Members/prices not loading
‚úÖ Solution:
   1. Check that data/ folder exists
   2. Check member_name.txt and fat_price.txt are present
   3. Set Working directory correctly
