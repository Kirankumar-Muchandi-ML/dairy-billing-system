STEP 2: INSTALL THE APPLICATION (ONE TIME ONLY)
================================================================================

ü™ü FOR WINDOWS:
   1. Double-click the file: INSTALL.bat
   2. Wait for installation to complete
   3. Press any key to close

üêß FOR LINUX:
   1. Right-click in the folder and select "Open Terminal"
   2. Type: ./install.sh
   3. Press Enter and wait

================================================================================
STEP 3: RUN THE APPLICATION (EVERY TIME YOU WANT TO USE IT)
================================================================================

ü™ü FOR WINDOWS:
   1. Double-click: START_DAIRY_APP.bat
   2. A black window will open (don't close it!)
   3. Your web browser will automatically open the application
   4. If browser doesn't open, go to: http://127.0.0.1:5000

üêß FOR LINUX:
   1. Double-click: start_dairy_app.sh
   2. Or open terminal and type: ./start_dairy_app.sh
   3. Open your web browser and go to: http://127.0.0.1:5000

================================================================================
STEP 4: USE THE APPLICATION
================================================================================

1. Select Member Name from dropdown
2. Select Milk Type (Cow or Buffalo)
3. Select Fat Content (%)
4. Enter Quantity in Liters
5. Click "Calculate Price"
6. View the bill
7. Click "Print Bill" to print or save as PDF

================================================================================
STEP 5: STOP THE APPLICATION
================================================================================

ü™ü FOR WINDOWS:
   Close the black window (command prompt)

üêß FOR LINUX:
   Press Ctrl+C in the terminal

================================================================================
HOW TO ADD/EDIT MEMBERS
================================================================================

1. Go to the "data" folder
2. Open "member_name.txt" with Notepad (Windows) or Text Editor (Linux)
3. Add member names (one per line)
4. Save the file
5. Restart the application

Example:
Rajesh Kumar
Priya Sharma
Amit Patel

================================================================================
HOW TO CHANGE PRICES
================================================================================

1. Go to the "data" folder
2. Open "fat_price.txt" with Notepad (Windows) or Text Editor (Linux)
3. Edit prices in format: fat%,price
4. Save the file
5. Restart the application

Example:
3.0,45
3.5,48
4.0,50
4.5,52
5.0,55
